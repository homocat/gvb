# 项目结构

```
.
├── api           接口目录
├── config        记录配置的结构体目录
├── core          初始化
├── docs          swag 文档
├── flag          命令行初始化
├── global        全局变量
├── middleware    中间件
├── models        表结构
├── routers       gin 路由
├── service       服务相关
├── settings.yaml 配置文件
├── testdata      测试文件
├── util          常用工具
├── main.go       入口
├── go.mod
└── README.md
```
